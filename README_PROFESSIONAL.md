# LicitaSuite 6.0 Professional

Versão profissional do LicitaSuite 6.0 Estável (LTS).

## O que esta entrega adiciona

- Arquivo de build profissional com PyInstaller.
- Ícone do aplicativo.
- Metadados do executável.
- Script para criar instalador com Inno Setup.
- Atalhos e estrutura de instalação.

## O que esta entrega não altera

- Motor de geração das Atas.
- Leitura do ZIP.
- Leitura do PDF.
- Leitura do Apêndice.
- Layout das Atas.
- Regras anti-invenção de dados.
