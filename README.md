# LicitaSuite 6.1 Professional FINAL

Pacote final de empacotamento profissional do LicitaSuite.

## Conteúdo

- `build_tools` — gera o executável `LicitaSuite.exe`;
- `installer` — gera o instalador com Inno Setup;
- `assets` — ícone do aplicativo;
- instruções de instalação.

## Observação

Este pacote não muda o motor das atas. Ele apenas transforma o projeto aprovado em aplicativo do Windows.
