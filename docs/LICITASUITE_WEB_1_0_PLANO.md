# LicitaSuite Web 1.0 Free

## Escopo

Criar acesso via navegador para o motor já aprovado do LicitaSuite.

## O que muda

- Interface passa a ser web.
- Usuário faz upload do ZIP.
- Sistema gera as atas.
- Usuário baixa o ZIP final.

## O que não muda

- Motor das atas.
- Leitura do PDF.
- Leitura do Apêndice.
- Formatação dos DOCX.
- Regras anti-invenção de dados.
