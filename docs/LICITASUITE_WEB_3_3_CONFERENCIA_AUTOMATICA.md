# LicitaSuite Web 3.3 — Conferência automática

Adiciona validação automática após a geração:

- Fornecedores encontrados no PDF;
- Fornecedores com DOCX gerado;
- Itens encontrados no PDF;
- Itens nas atas;
- Valor total lido do PDF;
- Valor total das atas;
- Divergências de fornecedores;
- Divergências de itens;
- Divergências de valores.

A conferência é adicionada ao ZIP final como:
RELATORIO_CONFERENCIA_AUTOMATICA.xlsx
