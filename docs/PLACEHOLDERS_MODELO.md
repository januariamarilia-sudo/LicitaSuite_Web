# Placeholders recomendados no modelo da Ata

Para preenchimento automático direto no texto do modelo, use:

- `{{FORNECEDOR}}`
- `{{RAZAO_SOCIAL}}`
- `{{CNPJ}}`
- `{{ENDERECO}}`
- `{{MUNICIPIO}}`
- `{{UF}}`
- `{{CEP}}`
- `{{TELEFONE}}`
- `{{EMAIL}}`
- `{{REPRESENTANTE}}`
- `{{CPF_REPRESENTANTE}}`
- `{{RG_REPRESENTANTE}}`
- `{{VALOR_TOTAL}}`
- `{{VALOR_TOTAL_EXTENSO}}`

Se não houver placeholders, o sistema preserva o modelo e acrescenta o quadro de itens ao final.
