# LicitaSuite Web 3.9 — Formatação final de quantidade e assinatura

Esta versão aplica dois ajustes finais no DOCX:

## 1. Quantidade com ponto de milhar

Exemplos:
- 3570497 → 3.570.497
- 204255 → 204.255
- 10653800 → 10.653.800

## 2. Assinatura

Padroniza linhas de assinatura para:
- Arial 11;
- negrito;
- primeira letra maiúscula.

A correção é aplicada de forma conservadora apenas em linhas curtas que pareçam assinatura.
