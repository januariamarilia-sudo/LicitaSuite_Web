# Status da v2.0

A v2.0 é a primeira versão limpa do LicitaSuite.

## Funcional
- Interface;
- Seleção de ZIP;
- Extração;
- Detecção de arquivos;
- Leitura do Apêndice;
- Leitura do PDF;
- Cruzamento de dados;
- Relatórios;
- Geração de DOCX por fornecedor;
- ZIP final.

## Próxima melhoria
Mapeamento específico do modelo oficial para preencher diretamente as tabelas existentes,
sem adicionar anexo ao final.
