# Motor DOCX Profissional

Este módulo prepara o LicitaSuite para editar o modelo real da Ata.

Princípios:

1. Nunca recriar o documento do zero.
2. Abrir o modelo original.
3. Alterar apenas dados variáveis.
4. Preservar cabeçalho, rodapé, margens, fontes, estilos e tabelas.
5. Informações ausentes devem ser destacadas nas próximas versões.
