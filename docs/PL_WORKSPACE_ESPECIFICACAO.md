# Especificação — Central de Processos

## Objetivo

Transformar cada Processo Licitatório em um workspace único, reunindo documentos, atas, histórico, status e relatórios.

## Estrutura do processo

- Dados Gerais
- Documentos
- Atas
- Habilitação
- Publicações
- Contratos
- Cronograma
- Relatórios
- Histórico

## Status do processo

- Em criação
- Arquivos recebidos
- Em processamento
- Atas geradas
- Com pendências
- Concluído
- Arquivado

## Status por módulo

- Verde: concluído
- Amarelo: pendente
- Vermelho: erro
- Cinza: não iniciado

## Integração com geração de atas

A Central de Processos apenas recebe os resultados do motor homologado:

- ZIP das atas;
- relatório financeiro;
- lista de fornecedores;
- lista de itens;
- divergências encontradas;
- data e hora da geração.

Nenhuma regra de geração das atas deve ser alterada.
