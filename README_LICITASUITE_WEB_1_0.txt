LICITASUITE WEB 1.0 FREE

OBJETIVO
Transformar o LicitaSuite em uma versão acessada pelo navegador.

IMPORTANTE
Este pacote NÃO altera o motor das atas.
Ele cria apenas a interface web.

COMO APLICAR

1. Faça uma cópia da sua pasta LicitaSuite 6.1 funcionando.

Exemplo:
C:\Users\janua\Documents\LicitaSuite_Web_1_0

2. Extraia este ZIP.

3. Copie a pasta:
web

4. Cole dentro da pasta:
LicitaSuite_Web_1_0

A pasta deve ficar assim:

LicitaSuite_Web_1_0
├── main.py
├── requirements.txt
├── licitasuite
├── config
├── docs
└── web

COMO RODAR LOCALMENTE

1. Entre na pasta:
web

2. Dê dois cliques em:
rodar_web_local.bat

3. Acesse:
http://localhost:8501

4. Envie o ZIP do processo.

5. Clique em GERAR ATAS.

6. Baixe o ZIP final com as atas.

PUBLICAR ONLINE GRÁTIS

A versão inclui:
- Dockerfile
- render.yaml

Eles serão usados depois para publicar gratuitamente em serviço compatível com Docker.
